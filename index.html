Layered Bars

The following styles are with a text overlay. The text overlay can easily be excluded by leaving the internal span element blank.

These bars are based on the css layout provided by Chris Coyier @ CSS-Tricks and modified by Jeff Langdon @ JL Design Network.

Include the jquery library, the included js and css files into the head of your html. 
<link href="src/style.css" rel="stylesheet">
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.5.0/jquery.min.js"></script>
<script type="text/javascript" src="src/script.js"></script>
